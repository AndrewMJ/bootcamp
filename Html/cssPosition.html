CSS Position Property has 4 possible values

  1. Static
  2. Relative
  3. Absolute
  4. Fixed

Static

Static is the default position value

Top, left, right, bottom will NOT work with static

A static element will show up exactly where it is in the HTML, left to right, top to bottom...it CANNOT be moved

Static Box

Relative Position

A relative postion is the same as static except it will honor top, left, bottom, right

if you move it, it will sit on top of other stuff on the page, and the page will think it;s still where it used to be

Absolute Position

Absolute elements work off the grid system. Just like fixed, they are ignored by EVERYONE.

They honor z-index

Like fixed, they do not leave gaps in the page

The top/left/right/bottom will be relative to the FIRST parent that has a position set

Fixed Position

A fixed position element will remain relative to the viewpoint (the screen) even if the user scrolls

Top, bottom, left, right will work just fine

Unlike relative, it will not reserve its old space. anything with static or relative will not know it's there.
